# ScreenShare


To run the SS Tool, copy and paste this command into cmd:

<" powershell Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass && powershell Invoke-Expression (Invoke-RestMethod https://raw.githubusercontent.com/Quickpots/Screenshare/main/SSTool.ps1) ">
